# Appointment Booking System

## Overview

This n8n workflow automates the appointment booking process by creating Google Calendar events and storing appointment details in Google Sheets.

## Tools Used

* n8n
* Google Calendar
* Google Sheets
* Webhooks

## Workflow

Webhook → Google Calendar → Google Sheets

## Features

* Automatic appointment booking
* Google Calendar event creation
* Appointment data storage in Google Sheets
* Real-time workflow automation
* No manual scheduling required

## Use Case

Ideal for consultants, agencies, coaches, lawyers, and businesses that want to automate appointment management and reduce manual work.
